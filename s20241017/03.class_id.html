CSS的選擇棄

CSS 有非常多選擇器.幫助我們在鑽寫樣式時,依照該情境選擇到我們要的目標

class選擇器

特性1: 舉有重複使用的特性,是使用率最高的選擇器

超連結

特性2:

抽離相同的屬性寫成class重複使用,再對不一樣的屬性定義class

class屬性是可以一值增加, 例如 box bg-color1



id 選擇器

一份文件裡面, 只允許一個獨一無二的id

對元素(標籤)設定id,可以用程式快速取得他

網頁區塊名稱會用id來命名
